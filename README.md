# Python Security Tools

A collection of Python-based security tools built to strengthen my understanding of cybersecurity concepts while improving Python development skills.

These tools demonstrate practical implementations of common security tasks including password generation, network scanning, hashing, log analysis, and file integrity verification.

---

# Tools Included

| Tool | Description | Status |
|------|-------------|--------|
| Password Generator | Generate strong secure passwords | 🔄 In Progress |
| Port Scanner | Scan TCP ports on a target host | ⏳ Planned |
| Hash Generator | Generate MD5, SHA1 and SHA256 hashes | ⏳ Planned |
| File Integrity Checker | Verify file integrity using SHA256 | ⏳ Planned |
| Log Analyzer | Analyse log files for suspicious activity | ⏳ Planned |
| Banner Grabber | Retrieve service banners from open ports | ⏳ Planned |
